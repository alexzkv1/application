# FASTAPI + React

Small end‑to‑end CRUD application for a single entity

# How to run: 
## Backend
```
cd backend 
python3 -m venv venv
pip install -r requirements.txt
cd app
fastapi dev main.py
```

Dont forget to create .env file with connection to your database  --> DATABASE_URL

## Frontend
```
cd frontend
yarn
yarn dev
```
